# Dorothea Rescue Centre - Admission Form System
## Simple Guide for Non-Technical Users

**Date:** January 2024  
**Project:** Digital Admission Form System  
**Purpose:** Replace paper forms with an easy-to-use computer system

---

## What Is This System?

This is a computer program that helps staff at Dorothea Rescue Centre fill out and manage child admission forms. Instead of using paper forms, everything is done on a computer, making it easier to:
- Fill out forms
- Find information quickly
- Print professional forms
- Keep records organized

---

## What Can You Do With This System?

### 1. Create New Admission Records
- Fill out a step-by-step form on the computer
- The form guides you through all the information needed
- Save everything with one click

### 2. View All Information
- See all details about any child in organized tabs
- Information is easy to read and well-organized
- Everything is in one place

### 3. Edit Existing Records
- Update information if something changes
- All previous information is saved
- Make changes easily

### 4. Download Professional PDF Forms
- Print a professional-looking form anytime
- The PDF looks just like the original paper form
- Includes all information and signatures

### 5. Search and Find Records
- Find any child's record quickly
- Search by name or other details
- See a list of all children

---

## How Does The Form Work?

The form is divided into 6 easy steps:

### Step 1: Child Information
- Child's name (first, middle, last)
- Nickname
- Gender
- Date of birth
- Ethnicity, religion, complexion
- Physical features

### Step 2: Rescue Details
- Where and when the child was found
- Who found them
- Case history (what happened)
- Previous places the child stayed

### Step 3: School Background
- Previous schools attended
- Current school information
- Education level

### Step 4: Family Information
- Parent or guardian details
- Sibling information
- Contact information

### Step 5: Health Status
- Hospitalization history
- Any illnesses
- Health notes

### Step 6: Signatures
- Three people sign the form
- Can upload signature images
- Date of signing

## How To Use The System

### Creating a New Admission:
1. Click "New Admission" button (green button)
2. Fill out Step 1: Child Information
3. Click "Next" to go to Step 2
4. Continue through all 6 steps
5. Click "Save Admission" when done

### Viewing a Record:
1. Find the child in the list
2. Click on their name
3. View all information in organized tabs
4. Click "Download PDF" to print the form

### Editing a Record:
1. Find the child in the list
2. Click "Edit" button
3. Make your changes
4. Click "Save" when done

## Questions & Answers

**Q: Is my information safe?**  
A: Yes! All information is stored securely and backed up automatically.

**Q: Can I still use paper forms?**  
A: You can print PDF forms anytime, but using the computer system is faster and more organized.

**Q: What if I make a mistake?**  
A: You can edit any record at any time to fix mistakes.

**Q: Can multiple people use this at the same time?**  
A: Yes! Multiple staff members can use the system simultaneously.

**Q: Do I need special training?**  
A: The system is designed to be easy to use. Basic computer skills are all you need.

**Q: Can I access this from home?**  
A: If the system is set up for remote access, yes. Otherwise, it's available on office computers.
